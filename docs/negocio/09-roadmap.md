## Roadmap breve

- **Corto (2 semanas)**
  - Import previsión: detección automática de columna desayunos (A/W fallback) + validación UI.
  - Calendario eventos: tooltips con pax y tipo; filtrado por salón.
  - OCR menús/recetas: agregar feedback de progreso.

- **Medio (1-2 meses)**
  - Autenticación real y roles.
  - Persistencia Supabase para todos los módulos (productos, recetas, lotes).
  - Recepciones y compras integradas con inventario.
  - Exportables PDF/Excel de hojas de producción.

- **Largo (3-6 meses)**
  - App móvil offline para tareas/turnos.
  - Modelos de predicción (desayunos, compras) con histórico.
  - Integraciones PMS/POS para sincronizar reservas y ventas.
